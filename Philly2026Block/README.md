# Philly2026Block

Interactive Philadelphia Marathon 2026 training block app.

## Deploy to GitHub Pages

1. Upload these files to your repo.
2. Go to **Settings → Pages**.
3. Set source to **Deploy from a branch**.
4. Choose **main / root**.
5. Visit your GitHub Pages URL.

## Custom domain

Add `philly2026.run` in GitHub Pages custom domain settings, then point your DNS to GitHub Pages.
